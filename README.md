# E-Commerce Dashboard Using Power BI

## Project Overview

The **E-Commerce Dashboard** is a comprehensive business intelligence tool designed to deliver real-time insights into various aspects of e-commerce operations. This project leverages **Microsoft Power BI** to integrate data, perform in-depth analysis, and create visually compelling, interactive dashboards. The dashboard is built to enhance decision-making by providing access to key metrics, product performance data, and employee insights, all in one centralized platform.

## Key Features

1. **Overview Page**  
   A concise display of essential Key Performance Indicators (KPIs) that provide an at-a-glance understanding of the business's overall performance. This page also includes brief explanations of the metrics to ensure clarity for users.

2. **Sales Page**  
   A deep dive into business metrics with interactive tooltips that offer a user-friendly experience and enable users to access key sales data easily.

3. **Product Page**  
   Detailed insights into product performance, allowing users to explore the data and uncover trends. The page empowers users to make informed decisions on product-related metrics.

4. **Employee Page**  
   An in-depth analysis of workforce data, showcasing performance, staffing levels, and more. Tooltips make it easy to interpret data at a glance, ensuring better workforce management decisions.

5. **Drilldown Page**  
   Focused on tracking product performance, the Drilldown page allows users to continuously monitor KPIs with access to detailed data, empowering real-time decision-making.

6. **Hidden Slicers**  
   Standard and hidden slicers enable users to filter and manipulate data easily, offering enhanced flexibility for data exploration.

7. **Interactive Tooltips**  
   Custom tooltips provide additional context for key metrics, improving the overall user experience and making the data more accessible and understandable.

## Technical Details

- **ETL Processes**: I conducted efficient **Extract, Transform, Load (ETL)** processes to ensure seamless data integration, real-time updates, and a smooth user experience on the Power BI dashboard.
  
- **DAX Functions**: Advanced **DAX** functions were used to create calculated fields and hidden cards, delivering precise analytics and insightful KPIs.

- **Row-Level Security (RLS)**: Implemented **Row-Level Security (RLS)** to secure data access, ensuring that over **10+ user roles** could access only the data relevant to their positions while maintaining strict data privacy.

- **Power BI Service**: Configured the **Power BI Service** for deployment, enabling automated **weekly refreshes** and reducing manual reporting time by **50%**.
